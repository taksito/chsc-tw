星際公民 繁體中文

Star Citizen Traditional Chinese Localization

1

獲取檔案

前往 GitHub 專案頁面，點選右上角的綠色按鈕 Code，接著選擇 Download ZIP 下載最新版本。

前往 GitHub 倉庫
2

解壓縮檔案

打開下載的 ZIP 壓縮檔，進入資料夾 starcitizen-chinese_traditional-main，你會看到以下內容：

  • data 資料夾
  • user.cfg 檔案
3

安裝至遊戲目錄

將上述的 data 資料夾 與 user.cfg 檔案 複製並貼上到你的星際公民遊戲安裝路徑中的 LIVE 資料夾下。

一般預設路徑參考：

C:\Program Files\RSI\StarCitizen\LIVE\
4

完成安裝

重新啟動遊戲，即可享受繁體中文介面。若更新後失效，請重新執行上述步驟覆蓋檔案。
